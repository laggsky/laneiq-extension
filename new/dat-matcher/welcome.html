Set Up Your Email Template

LaneIQ adds a one-click email button to every load row on DAT. When clicked, it opens Gmail with a pre-filled message to the broker — ready to send in seconds.

1
Click any highlighted load row on DAT to open the rate panel
2

3

2

3

4
Go to Credentials → Create API Key → copy it
5
Paste the key into the LaneIQ popup under Settings → Google Maps API Key
Free for normal use. Google Maps API has a generous monthly free tier — you won't be charged for typical dispatcher usage.
